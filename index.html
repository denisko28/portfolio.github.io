  • Solar Monitor

    HTML, JS, CSS + embeded C/C++ (STM32, ESP8266)

    The purpose of this project was to create a device designed to gather statistics on solar power generation from solar panels. This device establishes communication with a Schneider solar inverter through the RS485 port. The core of the device revolves around the STM32 microcontroller, which collaborates with the ESP8266, functioning as a WiFi module.


    I developed firmware for the STM32 microcontroller, it shows data on the LCD screen, allows user to navigate through menu and sends requests to the inverter every hour in order to collect fresh data. Once the data is received, the STM32 sends a message to the ESP8266 via the UART bus. I also wrote firmware for the ESP8266 module, enabling it to receive messages from the STM32 and forward them to my server. On the server side, I utilized PHP and a MySQL database.


    In addition, a web client has been created to visually represent the statistics on a diagram. Users have the ability to select their desired time period and observe the generated power statistics.

    Tech. stack used in project:
    HTML, JS, CSS, PHP, MySQL, embeded C, embeded C++.
